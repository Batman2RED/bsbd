## Курсовая работа по предмету "Безопасность систем баз данных"
##### Описание:
В курсовой работе рассматривается создание базы данных, предназначенной для автоматизации работы служб библиотеки. База данных позволяет клиентам просматривать ассортимент, краткое содержание. Также реализована автоматизация оформления заказа, которая позволяет клиентам формировать заказ.
